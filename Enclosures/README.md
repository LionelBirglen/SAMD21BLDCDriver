## Enclosures to 3D print for the driver and GM3506/4108 motors

### GM3506:
Print one of each of SAMD21_GM3506_xxx.stl files<br/>
Assemble with 4 M2.5x12-14 (pcb to motor+ring+cap) and 4 M2x25 flat head (previous assembly to tube) screws<br/>
See SAMD21_GM3506_assembly.png for a picture.

### GM4108:
Print one of each of SAMD21_GM4108_xxx.stl files<br/>
Assemble with 4 M3x8 (pcb to motor+ring), 4 M2.5x12 (previous assembly to cap), and 4 M3x8-20 flat head (previous assembly to tube) screws<br/>
Notice that there is a 15 degree shift between the pcb and ring part. This is required because the threaded holes on the GM3506 and GM4108 motors are different. So to use the same pcb for both motors, the latter needs to be slightly rotated.<br/>
See SAMD21_GM4108_assembly.png for a picture.
